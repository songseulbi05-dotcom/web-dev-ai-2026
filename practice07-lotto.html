로또 번호

  • 1 ~ 45 숫자 중 중복 없이 6개 + 보너스 1개
  • 1등 당첨 : 6개 일치
  • 2등 당첨 : 5개 일치 + 보너스
  • 3등 당첨 : 5개 일치
  • 4등 당첨 : 4개 일치
  • 5등 당첨 : 3개 일치
  • 내가 입력한 번호도 6개 숫자 랜덤으로 맞출 때까지
  • 힌트 : 기존 배열에 값이 포함되어 있는지 - includes() 사용
  • 테스트 : while(true) 무한루프로 인해 break는 넣고 위에서부터 코드 작성
  • 테스트 : node practice07-built-in-lotto.js 를 터미널에서 실행 가능
  • 테스트 : 해당 파일 폴더로 이동 폴더 들어가는 명령어는 'cd 들어갈폴더명'
